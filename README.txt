📦 УСТАНОВКА НА ХОСТИНГ
───────────────────────────────────────────────────

1️⃣ Требования к хостингу:
   ✅ Любой веб-хостинг с поддержкой статических файлов
   ✅ PHP 7.4+ (опционально, для будущей бэкенд части)
   ✅ MySQL/MariaDB (опционально, для базы данных)
   ✅ SSL сертификат (рекомендуется)

2️⃣ Загрузка файлов:
   
   Через FTP (FileZilla, WinSCP):
   ────────────────────────────────────────────────
   1. Подключись к своему хостингу по FTP
   2. Перейди в корневую папку (обычно /public_html или /www)
   3. Загрузи ВСЕ файлы из этой папки:
      - index.html
      - style.css
      - script.js
      - admin-setup.html
   
   Через панель хостинга (cPanel, ISPManager):
   ────────────────────────────────────────────────
   1. Открой Файловый менеджер
   2. Перейди в public_html
   3. Нажми "Загрузить" и выбери все файлы

3️⃣ Настройка домена:
   ────────────────────────────────────────────────
   1. В панели хостинга привяжи домен к папке public_html
   2. Подожди обновления DNS (до 24 часов)
   3. Сайт будет доступен по https://твой-домен.com

4️⃣ Создание админа (ПЕРВЫЙ ЗАПУСК):
   ────────────────────────────────────────────────
   1. Открой в браузере: https://твой-домен.com/admin-setup.html
   2. Введи данные администратора:
      - Username: admin
      - Email: admin@kalaxia.cc
      - Пароль: (минимум 6 символов)
   3. Нажми "Создать админа"
   4. Перейди на главную: https://твой-домен.com
   5. Войди через кнопку "Вход"

5️⃣ Проверка работы:
   ────────────────────────────────────────────────
   ✅ Сайт открывается
   ✅ Регистрация работает
   ✅ Вход в аккаунт работает
   ✅ Админ-панель открывается
   ✅ Покупка подписки работает

Изменить цены можно в файле index.html:
───────────────────────────────────────────────────
Найди секцию <section id="pricing"> и измени:
- amount: цена (например, 40)
- period: период (например, /24 часа)

В файле script.js измени:
───────────────────────────────────────────────────
planDurations = {
    '1day': 1,      ← дней для 1 дня
    '7day': 7,      ← дней для 7 дней
    '14day': 14,    ← дней для 14 дней
    '30day': 30,    ← дней для 30 дней
    'lifetime': 99999
}
